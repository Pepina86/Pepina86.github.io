    Intentionnalité

    Une relation assymetrique

    Si un objecte dans la relation normale n'existe plus, il n'y a plus relation.

    Dans la relation intentionnelle n'est pas comme ça ; par Example : je désire quelque chose qui n'existe pas (encore)

    Problème intrinsèque

    1. Toute conscience est intentionnelle = conscience de qqch.

    2. S est conscient de la table.

    3. S est conscient de son Y1. (Expérience psychique)

    4. Y1 est intrinsèquement conscient

    5. Donc : être objet de conscience intentionnelle = ! Être intrinsèquement conscient.

    6. Brentano : Donc Y1 est conscient de Y1 de façon réflexive.
